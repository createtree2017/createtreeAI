CreateTree AI Utilities - Embedding Guide

This guide shows how to embed the CreateTree AI Utilities into your PHP website. You can embed the entire app or individual features.

1. Include the Embed Script

First, add the embed script to your PHP page:

PHP
HTML
<?php // Your PHP code ?> <!-- Add this before the closing body tag --> <script src="https://[YOUR-REPLIT-APP-URL]/embed.js"></script>
<!-- Add this before the closing body tag --> <script src="https://[YOUR-REPLIT-APP-URL]/embed.js"></script>

2. Embedding the Complete Application

To embed the entire application:

PHP
HTML
<?php // Your PHP code ?> <div id="createtree-app"></div> <script> document.addEventListener('DOMContentLoaded', function() { createTreeEmbed({ target: '#createtree-app', height: '700px', width: '100%', maxWidth: '450px' }); }); </script>

4. Configuration Options

The embed script accepts the following configuration options:

createTreeEmbed({ // Required target: '#element-id', // Target element selector or DOM element // Optional feature: 'LULLABY', // Feature to embed: 'LULLABY', 'MEMORY', 'SUPPORT' height: '700px', // Height of the iframe (default: '670px') width: '100%', // Width of the iframe (default: '100%') maxWidth: '450px', // Maximum width of the iframe (default: '450px') showBorder: true, // Whether to show a border (default: true) showShadow: true // Whether to show a shadow (default: true) });
